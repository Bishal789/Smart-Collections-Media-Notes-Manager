# Sample Folder Tree for Import Testing

This folder contains sample files for testing the recursive folder import functionality of the Smart Collections application.

## Structure
- `notes/`: Contains text files.
- `pdfs/`: Contains PDF files.

## Instructions
1. Add `.mp3` and `.mp4` files to the appropriate folders.
2. Use the application to import this folder tree and verify the functionality.

## Example
```
sample_folder_tree/
├── notes/
│   ├── note1.txt
│   ├── note2.md
├── pdfs/
│   ├── document1.pdf
│   ├── document2.pdf
```
